// 기능

서버 운영에 필요한 전부

서버를 처음 여는 사람도, 모드팩을 친구들에게 나눠주려는 사람도. 필요한 도구를 한 곳에 담았어요.

원클릭 서버 제어

서버 시작·정지·재시작을 버튼 하나로. 실시간 콘솔과 RAM·CPU·TPS 모니터링까지 한눈에 봅니다.

모드 자동 설치

Modrinth에서 바로 검색해 설치하고, 친구들에게는 GitHub 연동으로 모드팩을 자동 배포할 수 있어요.

플레이어 관리

접속자 확인, OP 부여, 킥·밴을 클릭으로. 접속 기록과 스킨 미리보기도 함께 제공합니다.

Java 자동 감지

설치된 Java를 자동으로 찾고, 없으면 앱에서 바로 설치. 버전 충돌로 고생할 일이 없어요.

서버 코어 다운로드

Paper, Purpur, Fabric 등 원하는 서버 코어를 앱에서 바로 받아 설치합니다.

간편한 설정 관리

server.properties를 메모장 없이 UI로 편집. 날씨·시간·게임모드도 버튼으로 조정해요.

지금 바로 시작하세요

설치하고 서버 폴더만 선택하면 끝. 회원가입도, 로그인도 필요 없어요.
